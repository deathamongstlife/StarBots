# s.toggleslow
Slow the chat<br/>

`time` is the time in seconds users must wait after sending a message<br/>
to be able to send another message.<br/>
 - Usage: `s.toggleslow [time=0]`
 - Restricted to: `MOD`
 - Checks: `server_only`
Extended Arg Info
> ### time: int = 0
> ```
> A number without decimal places.
> ```
