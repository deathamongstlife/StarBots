# s.activity
Base command for creating activity invite links.<br/>
 - Usage: `s.activity`
 - Checks: `server_only`
## s.activity bashout
Create an activity invite link for Bash Out<br/>
 - Usage: `s.activity bashout <channel>`
Extended Arg Info
> ### channel: discord.channel.VoiceChannel
> 
> 
>     1. Lookup by ID.
>     2. Lookup by mention.
>     3. Lookup by channel URL.
>     4. Lookup by name
> 
>
