# s.autoban
Add term to the auto-ban list.<br/>
 - Usage: `s.autoban <term>`
 - Restricted to: `MOD`
 - Checks: `server_only`
Extended Arg Info
> ### term
> ```
> A single word, if not using slash and multiple words are necessary use a quote e.g "Hello world".
> ```
