# s.move

 - Usage: `s.move <new_channel> <msg_ids>`
 - Restricted to: `MOD`
Extended Arg Info
> ### new_channel: discord.channel.TextChannel
> 
> 
>     1. Lookup by ID.
>     2. Lookup by mention.
>     3. Lookup by channel URL.
>     4. Lookup by name
> 
>     
> ### *msg_ids: int
> ```
> A number without decimal places.
> ```
