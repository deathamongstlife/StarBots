# s.sudo (Hybrid Command)
Use `s.su`, `s.unsu`, `s.sudo` and `s.sutimeout`.<br/>
 - Usage: `s.sudo <command>`
 - Slash Usage: `/sudo <command>`
 - Checks: `Sudo`
Extended Arg Info
> ### command: str
> ```
> A single word, if not using slash and multiple words are necessary use a quote e.g "Hello world".
> ```
