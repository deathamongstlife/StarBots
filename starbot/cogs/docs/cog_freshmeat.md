# s.freshmeat
Show the members who joined in the specified timeframe<br/>

`hours`: A number of hours to check for new members, must be above 0<br/>
 - Usage: `s.freshmeat [hours=24]`
 - Restricted to: `ADMIN`
 - Checks: `server_only`
Extended Arg Info
> ### hours: int = 24
> ```
> A number without decimal places.
> ```
